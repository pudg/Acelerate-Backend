Endpoints


To make queries to the provided api use the following endpoints.

/api/{% verbatim %}{platform}{% endverbatim %}/restaurant/{% verbatim %}{identifier}{% endverbatim %}/reviews

Returns a list of all reviews from the specified platform for the restaurant identifier.
Use the following link to view a list of all reviews for Napoli Pizza.

Ex.
http://127.0.0.1:8000/api/grubhub/restaurant/2058686/reviews



/api/{% verbatim %}{platform}{% endverbatim %}/restaurant/{% verbatim %}{identifier}{% endverbatim %}/worst

Returns a list of all items rated with less than three stars from the specified platform for the restaurant identifier.
Open the following link to view a list of all problematic items from Napoli Pizza.

Ex.
http://127.0.0.1:8000/api/grubhub/restaurant/2058686/worst



/api/{% verbatim %}{platform}{% endverbatim %}/restaurant/{% verbatim %}{identifier}{% endverbatim %}/best

Returns a list of all items with five-star rating from the specified platform for the restaurant identifier.
Open the following link to view a list of all problematic items from Napoli Pizza.

Ex.
http://127.0.0.1:8000/api/grubhub/restaurant/2058686/best
